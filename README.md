# Power BI 作品集 | Superstore销售分析仪表板

**项目1：零售销售全链路BI仪表板**（个人独立完成）

### 项目亮点
- 处理 **49,000+ 条** 真实零售交易数据
- 构建星型数据模型 + 时间智能DAX
- 核心指标：总销售额、利润率、YoY同比增长、平均订单金额
- 包含交互式KPI卡片、月度趋势折线图、美国州级销售地图、Top10子类别分析、动态切片器

### 技术栈
- Power Query 数据清洗
- DAX高级度量值（CALCULATE、SAMEPERIODLASTYEAR、DIVIDE等）
- 地图可视化 + 书签导航 + Tooltip
